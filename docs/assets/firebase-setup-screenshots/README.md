# Firebase setup screenshots

Firebase setup guide screenshots are handled in two folders.

## inbox

Put original screenshots in `inbox/`.

This folder is ignored by Git because screenshots may contain project IDs, email addresses, UID values, tokens, service account details, or other sensitive data.

## masked

Masked screenshots for documentation go in `masked/`.

Only commit images after confirming the following are hidden:

- Firebase project ID when it identifies a real project
- Email address
- UID
- FCM token
- service account name or key details
- private key
- local file path
- billing or organization information
