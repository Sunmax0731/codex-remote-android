## Known limitations

- Google Play distribution is out of scope for this release.
- Each user must complete Firebase setup before first use.
- Existing debug-signed installs may need uninstall before installing this signed release APK.
- PC wake from power-off is not included.
- Release artifact upload must exclude `key.properties`, keystore/JKS files, service account JSON, `config.local.json`, `google-services.json`, `.env`, logs, and Firebase debug logs.
- Known low/moderate Firebase SDK transitive dependency warnings remain; high/critical audit checks passed.
