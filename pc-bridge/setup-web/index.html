Codex Remote Setup

Prepare Firebase, register local setup files, and generate the Android setup QR.

Cloud Links

Open the services needed for project creation and deployment.

Firebase Console Create the project and register the Android app. Google Cloud Console Check billing, APIs, IAM, and service accounts. Cloud Functions API Enable APIs required by Firebase Functions. Firebase CLI Docs Install and authenticate the Firebase CLI.

Procedure

Follow these steps before scanning the QR on Android.

  1. Create a dedicated Firebase project.
  2. Enable Authentication and Anonymous sign-in.
  3. Create Firestore Database.
  4. Register Android app package: com.sunmax.remotecodex.
  5. Download google-services.json.
  6. Create a PC bridge service account JSON and keep it local only.
  7. Generate the QR and scan it from the Android setup screen.
  8. Deploy Firestore rules, indexes, and Functions when needed.
