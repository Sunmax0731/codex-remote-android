# Skill: Repository and Environment

## Steps

1. Check `flutter --version`.
2. Check `java -version`.
3. Check Android SDK availability.
4. Check Firebase CLI availability.
5. Create project directories:
   - `app/`
   - `pc-bridge/`
   - `firebase/`
   - `docs/`
6. Add local setup instructions.

## Validation

- `git status` is clean after committed setup.
- Missing tool output is captured in documentation.
- Secrets and generated local config are ignored.
