# Agents: Repository and Environment

## Repository Agent

- Create the project structure.
- Keep generated app code, PC bridge code, Firebase functions, and docs separated.
- Maintain `.gitignore` for Android, Flutter, Firebase, and local secrets.

## Environment Agent

- Verify local tools: Flutter SDK, Android SDK, Java, Firebase CLI, Node.js or chosen PC bridge runtime.
- Document missing tools and installation steps.

## Handoff

This phase is complete when the repository builds or clearly documents missing prerequisites.
