# Agents: QA

## Test Agent

- Own unit, widget, integration, and bridge tests.
- Define the end-to-end command lifecycle test.

## Device QA Agent

- Test on Xperia 1 III or equivalent screen profile.
- Test WiFi and cellular network paths.

## Regression Agent

- Confirm security rules, notification routing, and command ownership after changes.

## Handoff

This phase is complete when the MVP end-to-end flow passes and release blockers are filed as issues.
